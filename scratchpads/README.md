# Scratchpads

This folder is for planning notes created before coding.

Each GitHub issue should have one scratchpad file.

Recommended filename:

```text
issue-<number>-short-title.md
```

Recommended format:

```markdown
# Issue #<number>: <title>

Issue link:

## Problem Summary

## Acceptance Criteria

## Prior Art

Related scratchpads:
Related PRs:
Relevant files:

## Implementation Plan

1.
2.
3.

## Test Plan

- [ ] 
- [ ] 

## Progress Notes

## Final Test Results

## PR Link
```
